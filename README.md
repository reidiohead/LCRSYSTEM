## 📖 Overview

The **Local Civil Registrar Management System (LCRMS)** is a web-based records management system developed to assist the Local Civil Registrar Office in organizing, storing, and managing civil registry records efficiently.

This project was designed and developed as part of my internship at the **Municipal Planning and Development Coordinator (MPDC) Office** of the Municipality of Rosario, Cavite during **July – August 2025**.

The system aims to streamline the management of civil documents, reduce manual record handling, and improve accessibility of archived records.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/lcr-management-system.git
```

2. Navigate to the project folder

```bash
cd lcr-management-system
```

3. Install dependencies

```bash
composer install
```

4. Configure environment file

```bash
cp .env.example .env
```

5. Generate application key

```bash
php artisan key:generate
```

6. Configure database credentials in `.env`

7. Run migrations

```bash
php artisan migrate
```

8. Start the development server

```bash
php artisan serve
```
